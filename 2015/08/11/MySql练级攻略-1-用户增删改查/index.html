MySql练级攻略(1)-用户增删改查

  • mysql

把自己的sql语句的练级过程记录下来，以mysql数据库为例。
（命令不区分大小写）

登录MYSQL:

cmd>mysql -u root -p
cmd>密码

查看：

MYSQL将用户信息存储在mysql.user中
可以使用select命令查看user表中的数据（当前用户必须有对user的查看权限）

mysql>select user,host,password from mysql.user;//或者使用以下两个分步语句
mysql>show databases;                      //显示当前用户可查看的所有数据库
mysql>use mysql;                           //选择mysql数据库
mysql>select user,host,password from user; //从user表中查询
mysql>select user();                        //查看当前用户

创建：

mysql>create user test identified by "123" //纯文本123加密存储在user表中password字段

修改：

mysql>rename user test to test1;

mysql>set password for test = password('123')//对user表没有操作权限的用户可以用此命令修改自己的密码
mysql>update mysql.user set password = password('123') where user = "test" //修改任意用户密码

删除:

mysql>drop user test;  //删除名为test的user

mysql>delete from mysql.user where user = "test" //不知道和上面的有什么区别
