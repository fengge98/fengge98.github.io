Vim练级攻略

发表于 2015-09-02   |  

vim的学习曲线相当的大（参看各种文本编辑器的学习曲线），所以，如果你一开始看到的是一大堆VIM的命令分类，你一定会对这个编辑器失去兴趣的。下面的文章翻译自《Learn Vim Progressively》，我觉得这是给新手最好的VIM的升级教程了，没有列举所有的命令，只是列举了那些最有用的命令。非常不错。

——————————正文开始——————————

你想以最快的速度学习人类史上最好的文本编辑器VIM吗？你先得懂得如何在VIM幸存下来，然后一点一点地学习各种戏法。

Vim the Six Billion Dollar editor

Better, Stronger, Faster.

学习 vim 并且其会成为你最后一个使用的文本编辑器。没有比这个更好的文本编辑器了，非常地难学，但是却不可思议地好用。

我建议下面这四个步骤：

1.存活
2.感觉良好
3.觉得更好，更强，更快
4.使用VIM的超能力
5.当你走完这篇文章，你会成为一个vim的 superstar。

在开始学习以前，我需要给你一些警告：

学习vim在开始时是痛苦的。
需要时间
需要不断地练习，就像你学习一个乐器一样。
不要期望你能在3天内把vim练得比别的编辑器更有效率。
事实上，你需要2周时间的苦练，而不是3天。

启 动Vim后，vim在 Normal 模式下。

现在，你知道如何在 Insert 和 Normal 模式下切换了。下面是一些命令，可以让你在 Normal 模式下幸存下来：

i → Insert 模式，按 ESC 回到 Normal 模式.
x → 删当前光标所在的一个字符。
:wq → 存盘 + 退出 (:w 存盘, :q 退出) （陈皓注：:w 后可以跟文件名）

你能在vim幸存下来只需要上述的那5个命令，你就可以编辑文本了，你一定要把这些命令练成一种下意识的状态。于是你就可以开始进阶到第二级了。

当是，在你进入第二级时，需要再说一下 Normal 模式。在一般的编辑器下，当你需要copy一段文字的时候，你需要使用 Ctrl 键，比如：Ctrl-C。也就是说，Ctrl键就好像功能键一样，当你按下了功能键Ctrl后，C就不在是C了，而且就是一个命令或是一个快键键了，在VIM的Normal模式下，所有的键就是功能键了。这个你需要知道。

第四级 – Vim 超能力

你只需要掌握前面的命令，你就可以很舒服的使用VIM了。但是，现在，我们向你介绍的是VIM杀手级的功能。下面这些功能是我只用vim的原因。

在当前行上移动光标: 0 ^ $ f F t T , ;
0 → 到行头
^ → 到本行的第一个非blank字符
$ → 到行尾
g_ → 到本行最后一个不是blank字符的位置。
fa → 到下一个为a的字符处，你也可以fs到下一个为s的字符。
t, → 到逗号前的第一个字符。逗号可以变成其它字符。
3fa → 在当前行查找第三个出现的a。
F 和 T → 和 f 和 t 一样，只不过是相反方向。

字符编码详解

发表于 2015-08-19   |  

基本概念

  • 字符是各种文字和符号的总称.

  • 字符集是字符的集合:，不同字符集包含的字符个数不同。常见字符集有：ASCII字符集、ISO8859字符集、GB2312字符集、BIG5字符集、GB18030字符集、Unicode字符集等.

使用哪些字符。也就是说哪些汉字，字母和符号会被收入标准中。所包含“字符”的集合就叫做“字符集”。规定每个“字符”分别用一个字节还是多个字节存储，用哪些字节来存储，这个规定就叫做“编码”。

各个国家和地区在制定编码标准的时候，“字符的集合”和“编码”一般都是同时制定的。因此，平常我们所说的“字符集”，比如：GB2312, GBK, JIS 等，除了有“字符的集合”这层含义外，同时也包含了“编码”的含义。

  • 编码：所谓编码就是用数字表示字符，例如用D7D6表示“字”。编码是将字符转化为数字的约定，规则。

  • UNICODE(UCS,Unicode Character Set)字符集为各种语言的每一个字符设定了统一并且唯一的数字编号，其他的编码方式通常是针对某一区域的语言，文字设计，只支持特定的语言。

  • 代码页：规定了适用于特定区域的字符集，以及编码方式。可以理解为字符和字节数据的映射表。不同代码页的编号可以理解为内码，如936为简体中文GBK，950为正体中文Big5。

  • utf(UCS Transformation format):UCS只规定了如何编码，并没有规定这些编码如何保存，传输。于是就有了utf，可以理解为utf是基于UCS的编码方案
    UTF-8就是以8位为单元对UCS进行编码。从UCS-2到UTF-8的编码方式如下：

UCS-2编码(16进制) UTF-8 字节流(二进制)
0000 - 007F 0xxxxxxx
0080 - 07FF 110xxxxx 10xxxxxx
0800 - FFFF 1110xxxx 10xxxxxx 10xxxxxx

例如“汉”字的Unicode编码是6C49。6C49在0800-FFFF之间，所以肯定要用3字节模板了：1110xxxx 10xxxxxx 10xxxxxx。将6C49写成二进制是：0110 110001 001001， 用这个比特流依次代替模板中的x，得到：11100110 10110001 10001001，即E6 B1 89。

总结

UCS为所有的语言的所有字符分配了统一并唯一的数字，UCS结合代码页对现在已经流行起来的一些编码方式做出支持。
文字实际上是以上述表格中utf-8的形式保存在系统中。当我们打开这个文件时(以utf-8为例)：
1.文字首先以utf-8的编码格式保存在文件中。
2.Windows在读取文件时通过BOM确认编码方式为utf-8，以选择对应的代码页65001。
3.按照编码规则 utf-8 -> unicode
4.Windows按照Unicode在指定代码页中查找字体图像，画到窗口上。

SQL注入

等级：紧急

描述：SQL注入利用现有应用程序，将恶意的SQL命令注入到后台数据库引擎执行，它可以通过在Web表单构造恶意SQL语句得到一个存在安全漏洞的网站上的数据库，而不是按照设计者意图去执行SQL语句.
可以直接或间接的获取数据库内容，危害较大.

建议：

1.过滤客户端提交的敏感字符，提交的区域包括：地址栏中的所有参数，请求头信息，cookie中的参数，需要过滤的字符如下：
|–&–;–$–%–@–';
"–<–>–(;)–+–CR;
LF–,–.–select–from;
update–delete–union–and–or;
2.白名单策略：定制允许客户端输入的长度，类型.
3.做好数据库用户权限管理，只给用户开放必要权限.

XSS跨站脚本攻击

1.反射型xss
等级：低

描述：恶意攻击者往Web页面里插入恶意html代码，当用户浏览该页之时，嵌入其中Web里面的html代码会被执行，从而达到恶意攻击用户的特殊目的.
反射型XSS通过给别人发送带有恶意脚本代码参数的URL，当URL地址被打开时，特有的恶意代码参数被HTML解析、执行.
它的特点是非持久化，必须用户点击带有特定参数的链接才能引起.

2.存储型XSS
等级：中

描述：恶意攻击者往Web页面里插入恶意html代码，当用户浏览该页之时，嵌入其中Web里面的html代码会被执行，从而达到恶意攻击用户的特殊目的.
存储型XSS通过上传非法代码，当用户触发恶意代码，浏览器就执行攻击者定义的脚本程序.

建议：
1.对用户输入进行数据合法性验证:

不可信数据将被放置的地方 例子 应该采取的编码 编码格式
HTML标签之间 <div> 不可信数据 </div> HTML Entity编码 &–>&amp;<–>&lt;>–>&gt;”–>&quot;‘–>&#x27;/–>&#x2f;
HTML标签的属性里 <input type=”text”value=” 不可信数据 ” /> HTML Attribute编码 &#xHH;
JavaScript标签里 <script> var msg = ” 不可信数据 ” </script> JavaScript编码 \xHH
HTML页面的URL里 <a href=”/page?p= 不可信数据 ” >…</a> URL编码 %HH
CSS里 div style=” width: 不可信数据 ” > … </div> CSS编码 \HH

2.为Cookie加上HttpOnly标记:
许多XSS攻击的目标就是窃取用户Cookie.
窃取Cookie一般都会依赖JavaScript读取 Cookie信息，而HttpOnly标记的cookie不允许被JS操作，这样即使Web应用产生了XSS漏洞，Cookie信息也能得到较好的保护.

CSRF跨站请求伪造

等级：中

描述：攻击者盗用用户身份，以用户名义发送恶意请求.
CSRF可能造成的危害包括：以用户名义发送邮件，发消息，盗取账号，甚至于购买商品，虚拟货币转账，个人隐私泄露以及财产安全.

建议：
1.验证HTTP Referer字段：
验证每一个请求的Referer值，如果来自本站，是合法的.
如果Referer是其他网站的话，就有可能是CSRF攻击，则拒绝该请求.

2.在HTTP请求中以参数的形式(或是通过XMLHttpRequest这个类加到请求头中)加入一个随机产生的token，并在服务器端建立一个拦截器来验证这个token，如果请求中没有token或者token内容不正确，则认为可能是CSRF攻击而拒绝该请求.

clickjaking点击劫持

级别：低

描述：点击劫持是一种视觉上的欺骗手段.
攻击者使用一个透明的、不可见的iframe，覆盖在一个网页上，然后诱使用户在该网页上进行操作，此时用户将在不知情 的情况下点击透明的iframe页面.

赋值有如下三种：

1.DENY：不能被嵌入到任何iframe或者frame中.

2.SAMEORIGIN:页面只能被本站页面嵌入到iframe或者frame中.

3.ALLOW-FROM uri：只能被嵌入到指定域名的框架中.

</IfModule>

描述：

亲爱的：

十分抱歉，近日忙于新《讲义》的修改，回信稍有延迟，愧疚万分。分离二地无法执手相望，方知剪不断理还乱的不止离愁还有思念。时常于疲倦之时反复翻阅往日之来信，你的文字于书轩中伴我看遍月落日升，那份娴静恰如你似水之温柔。近闻得你在新环境下，生活略有不适，常为一些无脑之言论所扰，愧疚于无法此刻将你揽入胸怀以安慰，情急无奈之下又愁吐香烟、乱拍“栏杆”，现回信一封，说些愚见。

1.关于民族与国籍的问题：

亲爱的，我曾经有过和你同样的苦恼，所以我深刻地清楚，庸众问你国籍与民族等问题时你所忍受的尴尬。你是否和别人解释再多也解释不清楚，相信你曾和我有过同样的好奇，“为什么那么多中国人能理解自己的民族是“汉族”而不是“中国族”；却难以理解日本人是“大和民族”而不是“日本族”；韩国人是“朝鲜族”而不是“韩国族”。这个问题解释不清楚并不是你的过错，也没有必要为一些人的误识所纠结，我们只需要记住中国是我们的祖国，高丽是我们的民族即可。国籍，毕竟只是一种政治上的归属；民族，才是对血统和文化的认同。比如说，当我们在给学生们解析Korea这一词时，没必要强加“North Korea”和“South Korea”这种政治性的概念，把侧重点放在告诉学生们Korea是“고려——Ko Liao（高丽）”的音译岂不更好。虽然曾经统一的朝鲜由于政治性的原因如今分裂成了今日的“朝鲜”和“韩国”，而且韩国为了在意识形态上将其与北朝鲜区分开，给自己的人民和语言“重新”起了一个“可以迷惑外国人眼球”的名字——“大韩民族”和“韩国语”。 但无论是怎么改，在血脉上Korean对通古斯民族血脉的继承可是永远改变不了的啊。亲爱的，就像你我作为中国人所坚信的那样——不论当今的台独分子以何等卑劣的手段来破坏华夏的统一，台湾同胞永远是和我们一样的炎黄子孙、龙的传人。让一切的争论都融化在周恩来总理万隆会议上的那句发言里吧——“万物相育而不相害，道并行而不相悖”这正是庄子的智慧。

2.关于新的生活环境：

亲爱的，我知道，你成长于一个标准的朝鲜族家庭，从小到大一直是吃韩食、住韩居，甚至还有那种朝鲜族典型的小“洁癖”，但是突然来到了另一个民族的文化圈子，短时间要适应这些实属不易，就像是我刚到新环境时很多人无法理解我每天天要刷四次牙，至少洗三次澡一样。但是希望你能记住“生活并不只有眼前的苟且，还有诗和远方的田野”。我记得我读书时，我的一位中国老师曾拍着我的肩膀语重心长地说“苦不苦，想想红军两万五；累不累，想想革命老前辈”。是啊，每当疲惫之时就想想我们的祖先吧，他们为了让子孙后代能过上幸福的生活，是克服了多少苦难啊。

我们居住在朝韩半岛的祖先，想想就让人心疼，每每回望那片土地，总是有让人落几滴泪的冲动……

你想想啊，朝韩半岛多山而少平原的地形，并不适合农耕也不适合放牧；那里只有少数南部地区才算得上是亚热带季风性气候，大部分地区都是相对寒冷的中温带，不但耕地不足、庄稼的生长期更是短得可怜；那里没有华夏大地那样广袤的暖温带大平原、可供灌溉的汤汤大河；只有少量破碎的沿海平原，并且还要时常担心于海水倒灌、台风雨等等危机。 所以，我们的祖先面对的人口与粮食的矛盾，人口与土地的矛盾要比其他地区更紧张啊。为了生存，他们必须在十分辛勤、精细劳作的同时还要尽最大可能地充分利用每一寸土地。不过，在这样的环境下打拼，代代相传，自然磨练出了我们高丽人坚忍不拔、勤劳勇敢的意志品质以及高超的水稻种植技术。讲究中庸、好吃懒做、不善钻研的民族只适合呆在暖温带那样的平原安乐窝。在清朝后期，清政府施行“开关”的政策，我们的先民一批批的来到了中国东北地区开垦白山黑水之地，同时还要替清政府抵抗侵略中国东北的日本、沙俄士兵，这是有多难啊！但是，亲爱的，我们的先民并没有投降，而是鼓起勇气克服重重困难，反而培育出了驰名中外的东北大米的雏形。直到现在中国东北的一些农场里还可以遇到一些中国国籍的朝鲜族同胞，我曾和他们交谈过两句，他们的父辈恰恰是早些年教授汉民如何在白山黑水之地种植水稻的技术员。我曾经一直庆幸着，在我20多岁的生活经历中，一直都在吃着最擅长种植水稻的民族所种植的大米，而且也正是流淌着这个民族血液的母亲每日为我精心地烹制米饭；同时我也感谢命运，让我又遇到了你。亲爱的，请原谅我的贪婪，因为此刻的我已经在开始构想着你穿着韩服盘着束发烹制米饭时美丽的样子了。我相信经过你手烹制的米饭一定会有一份我们民族的那种精细，而这份能力一定就像我们继承了自身的高丽血统一般自然；那饭桌也定会是洁净万分的，因为能在多山而少平原地区种植出上等水稻的民族，一定会爱惜每一寸土地，一定会讲求精细，而有着这份精细的乡土概念的民族，怎会让居室内有一粒灰尘呢。

3.关于你此次“Gap Year”的决定是否正确：

汉人有一句俗语，我认为很有道理“出来混，早晚要还的”。所以虽然此刻的你在经受着“culture shock”所带给你的种种不解与不便，但我还是坚信你这次“Gap Year”的决定是正确的，而且终会使你受益万分的，因为我相信作为高丽儿女，你的骨子里就有一种“闯”劲。

你看哈，大凡是深刻受海洋文明影响的民族，为了生存都会有一种远征的欲望。大到轰轰烈烈的大航海时代，小到英国米字旗和日本国旗上那由一点滴而向外辐射的条纹。

我们的祖先，为了缓解生产与生活的矛盾，在朝韩半岛上不光要辛勤地农耕，还要在山林里进行采集、狩猎；同时“黄海暖流“和”西朝鲜寒流“在半岛的西侧汇聚；”对马暖流“和”库页岛寒流“在东部汇聚，此乃天然的上等渔场，我们的先民还要进行海洋捕鱼的作业。然而无论是山林中的采集、狩猎还是出海打渔，都要有一种打破小农庄稼院、向外勇于探索的气魄，这就是融在高丽儿女身上的那股“闯”劲儿。也正因如此，即便朝韩半岛的农业生产很重要，但先民们也并未禁锢于这狭小的农耕天地。在农耕性层面上，我们的民族有着中华儒家文化一套“温良恭俭让”的完美诠释，“重教育、守礼乐”传统文化内核；同时在需要冒险的领域内，作为通古斯后裔，骨子里所赋予的虎性血液又不断地激荡翻腾。也许正是这种重视教育又敢于冒险的民族性格成就了我们的民族。虽然我们民族的人口数仅占中国总人口数比例的”千分之一点”多，但却出了一大批学者，一大批艺术家。比如:既写了《中国人民解放军进行曲》又写成了《朝鲜解放进行曲》的정률성（郑律成）；开着低端战机却击落了四架美军高端战机的拼命将军—리영태（李永泰）；中国第一位影帝김염（金焰）；中国现代舞的奠基人——金星；喊出了中国摇滚第一嗓的摇滚教父——崔健；当然还有喜欢实话实说的罗永浩等。

亲爱的，每当我看到你在求知上的那份闯劲，对待学习的那份热情，我就会回想起季羡林老先生在《行走世界》那本书里的句子——“我国的朝鲜族十分重视文化、艺术教育，文化水平位列全国之首”；我也问过自己“为什么经数据表明虽然我们民族的姊妹弟兄绝大多数都居住在教育资源并不丰富的边疆地区，但是在56个民族中却是受教育程度是最高的，而且还是全国平均水平的两倍多？”然而当我久久伫足于延边朝鲜族博物馆时，亲爱的，那刻在墙上的一句朝鲜族俗语揭晓了答案——“即使挨饿，也要学习”… …

的确，这样想想我们的先民似乎又生活在了一片乐土。在一个深刻受到了儒家礼乐文化影响且历史上少有大规模知识分子惨遭屠杀事件的地方，人们自然对知识会有着一种近乎偏激的执着。

亲爱的，你还记得我之前给你写信，兴奋地告诉你我的一位朝鲜族的新学生么？她第一次在我家听完课准备回家之时，背对着我蹲在门口的地垫上温柔且可爱地穿好自己的鞋子，随后迅速地一转身把自己刚穿过的拖鞋十分整齐摆放到我家的鞋架上，她站起身深深一鞠躬，用非常标准的敬语体式对我说了一句“您辛苦了”才转身离去。于是那个假期我记忆中最美的画面就是每天讲完课后与她互相在门口鞠躬道别，道一声“再见，请慢走”她回一句敬语“好的，我会小心，谢谢您”，于是每天的那个时段，我都感觉仿佛我之前读到的所有“朝韩史”文字一下子都活了，轻轻跳动着，如她一般让人温暖欢畅。曾自称“礼仪之邦”的国度的群众，如今走到海外因为“没有礼仪、没有素质”而被好多国家所嫌弃；我很自豪地看到洋人眼中的那份东方礼仪如今是由Korean和Japanese表现得具体而又生动，融化在人与人鞠躬行礼的一个动作；升华在晚辈向长辈敬酒后要转身方可饮的礼节中；让人陶醉于长幼有序的声声敬语里。

4.关于你是否应该坚持爱好：

哦，亲爱的，我是一个粗人，我可不敢斗胆和你谈论艺术与修养。但是你既然抱怨到了学业与你音乐爱好间的矛盾关系，我想我还是有些必要谈谈自己于此的粗浅的见解，当然处于自私的角度，我也更喜欢婚后不看韩剧而是能教孩子弹琴唱歌的你。所以呢，我的建议是：无论再忙，都希望你能把自己的音乐爱好坚持下去。

我认为我们高丽人较之日本人是少了些许死板得，多了些灵活的；你热爱音乐，我想从这一点上你就能看出个一二。

我们民族的音乐，有很多和中国传统音乐的相同之处。她不同于大和民族的音乐有着许多的“3-4”“7-1”这种半音式组合或者“小三和弦”，它更多的是类似于中国传统的五声音阶：宫、商、角、徵、羽（1、2、3、5、6）。如果你将典型的朝鲜族歌曲《阿里郎》、《桔梗谣》等还原成C大调，那么我们不但发现不了“半音组合”而且连“变徵”（7）这个音都找不到。一些受欧洲性元素影响的音乐，如典型的俄罗斯音乐常以小调的形式出现，乐曲常为“降E”调，日本性音乐也不乏小调作品且多出现“半音组合”和变徵之音。所以单纯就这点而言，朝鲜民族的音乐和中国传统音乐还是更为亲近的。从典型的朝鲜族乐器中也可找出例证，比如说与竹笛相似的细筚篥；与古筝很相似的伽倻琴（我知道这是你的最爱O(∩_∩)O~）。此外，在欧洲性的音乐中常见3/4拍，6/8拍那样的乐曲，让人有种想跳舞的冲动；美国的乡村民谣也成了4/4拍节奏应用的典型，但是到了朝韩半岛，不光会有《阿里郎》《桔梗谣》这种婉转又悠扬的3/4拍乐曲，还会有《红太阳照边疆》《农村四季歌》、《春耕谣》、《苹果丰收》这样欢快的乡村气息歌曲，但是别忘了，我们的民族始终流淌着通古斯的血液呢、所以我们也并不缺少在2/4拍这种进行曲节奏上的出色表现。就如你父亲最尊敬的那位艺术家，中国军歌之父、中国朝鲜族作曲家정률성（郑律成）所写的《八路军进行曲》《延安颂》《中国人民解放军进行曲》《朝鲜人民军进行曲》一样铿锵。

亲爱的，你还记得那次我们一起听的爵士音乐会么？其实那天我一直在思考一个问题，而且越想越有趣。你看哈，1910年左右发源于美国佛罗里达州南部的爵士乐，在上个世纪六、七十年代传入日本和韩国，在那样一个部分亚洲国家疯狂亲美的年代，爵士乐毫无阻力地融入了日本的流行音乐中，不声不响如村上春树手中会拿着拉罐啤酒般自然；而到了韩国，爵士乐并非仅仅仍是老套的钢琴、贝斯和铜管乐器组合，而是融入了“长鼓”这一高丽特色。长鼓的声声敲击，似乎提醒着我们这些高丽儿女，即便是在享受舶来品爵士乐的同时，也别忘了身体里的民族血液，崇洋尚可，媚外不得！

我至今还记得我们一起听音乐会的那个雨夜，演出结束后你问我“乔，这次你为何这么安静？”我当时微笑着没有说话。哦！亲爱的，你可知道当时我心中的波澜！当时牵着你手的我早已神游至天际，那长鼓震颤的音腔恰如我当年独自一人在朝韩半岛寻根时澎湃的胸膛，那音符似乎带我重回了昔日时光，我早已随他们眼含热泪地翻越了一道道民族与爱的山岗。

5.关于我的近况：

亲爱的，你懂的，我还是那个不会开车，不会淘宝，连KTV都不去的无聊男人。至于说娱乐活动么，好多天前的一个晚上出门玩儿了一会儿，和“姜城子”看了一部叫《狼图腾》的电影。我一直在和“姜城子”探讨一个问题，虽然问题的答案目前还无法成为一个体系，但是已经稍微有了一丁点的眉目。

你看啊，农耕民族种植庄稼，有时要靠“天”吃饭，所以华夏文名所崇拜的自然就是能掌控四季风调雨顺的“龙图腾”；草原民族不能让野鼠、野兔和过量的羊群破坏草场，所以需要狼来控制草原的生态平衡，而且死后还要用狼来实行“天葬”，于是自然有了“狼图腾”；但是像高丽人以及其他居住山林地带常与野兽相伴的通古斯后裔，自然就有了对“虎图腾“的敬畏。结果前些日子十分荣幸在延边州游走时，无意地走入珲春市的一个公园，穿过了林荫的山路小道，一路上思虑着这曲径幽处会带给我何样的惊喜，结果我忽见一块巨石，上面浮雕着四头面朝四方通天咆哮的巨虎！亲爱的，那一刻一种骨子里的民族亲近感随着血液的喷涌蔓延至我全身，我震颤着摘掉了头上的帽子，冒着大东北近零下20度的寒风，仰天长啸一声。那份油然而生的敬慕，恰如阮籍与孙登以长啸代绵谈的对话，似乎万千言语都不如这一份玄妙，碰撞着我身上的高丽血统，恰如遥远的绝响。第二日我又乘车奔赴延吉的帽儿山，方罢下车又在那山腰处看到一巨虎图腾。华夏民族向来讲究坐北朝南从而便于采光，但是通古斯的分支大多喜欢朝东拜日，而那巨虎恰是面朝东方，似乎在守望着代代高丽儿孙。亲爱的，你可知道，当时我是有多希望你在我身边分享这一份喜悦啊！
